About Beer Analytics

What is Beer Analytics?

Beer Analytics is a database of 400,000+ beer brewing recipes, built for data analysis. It helps brewers and beer enthusiasts understand brewing trends and recipe composition.

Data Sources

This database aggregates publicly available brewing recipes from:

  • Brewtoad — 330,000+ preserved recipes (legacy archive)
  • BrewGr — 93,000+ preserved recipes (legacy archive)
  • Homebrewing Communities — ongoing recipe contributions
  • BJCP Standards — official beer style definitions
  • Hop Producer Data — Hopsteiner, Yakima Chief, BarthHaas, Crosby

India-Friendly

This is a minimal, India-specific version with metric units (liters, kilograms, Celsius) as the default. Toggle between metric and imperial units anytime using the button in the top navigation.
